# EcoWaste Management Website

This project is a simple EcoWaste Management website created using HTML, CSS, and JavaScript. The main aim of this project is to spread awareness about proper waste management and encourage people to keep the environment clean.

The website provides information about different types of waste and shows how waste can be managed or recycled properly. It also includes an interactive and user-friendly design to make learning about eco-friendly practices easier.

## Features
- Informational sections about waste management
- Simple and clean user interface
- Responsive design
- Interactive elements using JavaScript

## Technologies Used
- HTML
- CSS
- JavaScript

## Purpose
The purpose of this project is to promote environmental awareness and demonstrate basic web development skills.
